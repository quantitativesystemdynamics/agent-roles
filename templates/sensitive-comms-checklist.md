# Sensitive Communications Checklist Template

## Purpose
Pre-send checklist for communications that may have legal, `hr-generalist`, or reputational implications.

---

## Pre-Send Checklist

### Classification
- [ ] Is this communication confidential?
- [ ] Does it involve inside information?
- [ ] Does it require legal review before sending?
- [ ] Does it require `hr-generalist` review before sending?

### Content Review
- [ ] Are all facts verified and accurate?
- [ ] Are commitments realistic and achievable?
- [ ] Is tone appropriate for the audience?
- [ ] Are there any statements that could be misinterpreted?
- [ ] Is forward-looking language properly qualified?

### Regulatory Considerations
- [ ] Does this contain material non-public information?
- [ ] Are regulatory disclosures required?
- [ ] Is communication consistent with prior public statements?
- [ ] Is communication consistent with regulatory guidance?

### Recipient Review
- [ ] Is recipient list correct and complete?
- [ ] Are any recipients on the bcc list inappropriately?
- [ ] Is communication going to appropriate individuals only?
- [ ] Are there any conflicts of interest with recipients?

### Documentation
- [ ] Is communication appropriately documented?
- [ ] Is retention required? If so, where?
- [ ] Is there a clear record of approval?

---

## Approval Gates

### `general-counsel` Review Required
- Communications about litigation
- Communications about regulatory matters
- Communications about significant company events
- Communications about contracts or commitments

### `hr-generalist` Review Required
- Communications about personnel matters
- Communications about workplace issues
- Communications about benefits or compensation
- Communications about policies or procedures

### Communications Review Required
- Press releases and public statements
- Investor communications
- Customer-facing communications about material matters
- Employee communications about significant changes
